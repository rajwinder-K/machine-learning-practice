# Titanic Survival Prediction

## Project Overview

This project predicts whether a passenger survived the Titanic disaster using Machine Learning. It was created as a practice project to learn the complete machine learning workflow, from data exploration to model evaluation.

---

## Objectives

- Understand the Titanic dataset.
- Perform data cleaning and preprocessing.
- Explore the dataset using EDA.
- Train machine learning models.
- Compare model performance.

---

## Dataset

- Source: Kaggle Titanic Dataset
- Rows: 891
- Columns: 12

## Features

- PassengerId
- Survived (Target)
- Pclass
- Name
- Sex
- Age
- SibSp
- Parch
- Ticket
- Fare
- Cabin
- Embarked
